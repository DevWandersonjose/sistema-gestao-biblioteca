# sistema-gestao-biblioteca
Sistema de gerenciamento de biblioteca desenvolvido em Java com interface gráfica Swing para controle de livros, alunos e empréstimos.

Este é um sistema de gerenciamento de biblioteca desenvolvido em Java. O objetivo do projeto é aplicar conceitos de Programação Orientada a Objetos (POO), arquitetura em camadas e construção de interfaces gráficas desktop.

O sistema permite o cadastro de alunos e livros, além de gerenciar o fluxo de empréstimos e devoluções, persistindo todos os dados em arquivos locais (.csv), eliminando a necessidade de configuração de banco de dados para testes rápidos.
